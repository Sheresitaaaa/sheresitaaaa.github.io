# Portfolio Website

Digitale portfolio-website gebouwd voor het vak **Data Structures & Algorithms** (DSA 26).
Gemaakt met **HTML, CSS en JavaScript** — volledig responsive (desktop, tablet, mobiel).

## Pagina's

| Pagina | Bestand | Inhoud |
| --- | --- | --- |
| Home | `index.html` | Intro, huidige rollen, werkervaring, recente projecten, skills |
| Over mij | `about.html` | Achtergrond, opleiding, interesses |
| Werk | `work.html` | Projecten met titel, beschrijving, technologieën en link |
| Contact | `contact.html` | Contactgegevens + contactformulier |

## Projectstructuur

```
portfolio-website/
├─ index.html
├─ about.html
